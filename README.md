# Quincy Miller - Game Boy Landing Page

Retro Game Boy themed landing page with glitch effects.

## Features
- Authentic Nintendo Game Boy DMG-01 design
- VHS static and CRT scanlines
- Chromatic aberration glitch effects
- Screen burn simulation
- Interactive menu with hover effects
- Fully responsive
